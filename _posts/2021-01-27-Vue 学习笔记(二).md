<p align="center">双向数据绑定指令 v-model</p>
---

v-model 指令一般使用在表单元素上(文本框 多选框 单选框 文本域 下拉框等等)，进行双向数据绑定，一方发生变化，另外一方也会随着改变。v-model 使用在不同的表单元素上，绑定的类型也不一样。

1.v-model 作用于文本框

```
<body>
    <div id="app">
        <div>{ {msg} }</div>
        <input type="text" v-model="msg" />
    </div>
    
    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                msg: 'Hello World'
            }
        })
    </script>
</body>
```

2.v-model 作用于多选框

```
<body>
    <div id="app">
        <div>{ {checked} }</div>
        <input type="checkbox" v-model="checked" />
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                checked: true
            }
        })
    </script>
</body>
```

<p align="center">指令 v-text、v-html</p>
----

v-text 指令用于普通文本插入，将 data 中普通文本内容替换指定元素中的内容。v-html 指令用于 HTML 代码片段插入，将 data 中 HTML 代码片段替换指定元素中的内容。v-text 和 v-html 指令都是单项数据绑定。v-text 指令与插值表达式的区别在于，v-text 指令会将元素中的内容替换成指定数据，插值表达式只会将表达式所在位置替换成指定数据。

1.v-text 指令使用示例

```
<body>
    <div id="app">
        <div v-text="content"></div>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                content: 'Hello World'
            }
        })
    </script>
</body>
```

2.v-html 指令使用示例

```
<body>
    <div id="app">
        <div v-html="content"></div>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                content: '<h1>Hello World</h1>'
            }
        })
    </script>
</body>
```

3.v-text 与插值表达式

```
<body>
    <div id="app">
        <div>---{ {content} }---</div>
        <div v-text="content">------</div>
    </div>
    
    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                content: 'Hello World'
            }
        })
    </script>
</body>
```

<p align="center">属性绑定指令 v-bind</p>

----

v-bind 指令用于属性绑定，将指定的属性与 data 中的数据进行绑定，单项数据绑定。v-bind 指令能够作用于任何属性。该指令的简写形式为 :，使用示例如下：

1.v-bind 指令使用示例

```
<body>
    <div id="app">
        <a v-bind:href="href">百度搜索</a>
    </div>
    
    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                href: 'https://www.baidu.com'
            }
        })
    </script>
</body>
```

2.v-bind 指令简写示例

```
<body>
    <div id="app">
        <a :href="href">百度搜索</a>
    </div>
    
    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                href: 'https://www.baidu.com'
            }
        })
    </script>
</body>
```

v-bind 指令作用于 class 和 style 属性时，vue 做了专门额增强。属性值除了上述示例中属性值是一个字符串映射 data 的属性，还可以是对象或数组，示例如下：

3.v-bind 绑定 class 属性值是字符串

```
<style type="text/css">
    .style1 {
    	background-color: red;
    }
</style>

<body>
    <div id="app">
        <div :class="style">hello world</div>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                style: 'style1'
            }
        })
    </script>
</body>
```

4.v-bind 绑定 class 属性值是个对象

```
<style type="text/css">
    .style1 {
        background-color: green;
    }
    .style2 {
        color: red;
    }
</style>

<body>
    <div id="app">
        <div :class="{ style1 : isStyle1,style2 : isStyle2  }">hello world</div>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                isStyle1: true,
                isStyle2: true
            }
        })
    </script>
</body>
```

5.v-bind 绑定 class 属性值是个数组

```
<style type="text/css">
    .style1 {
        background-color: green;
    }
    .style2 {
        color: red;
    }
</style>

<body>
    <div id="app">
        <div :class="[isStyle1,isStyle2]">hello world</div>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                isStyle1: "style1",
                isStyle2: "style2"
            }
        })
    </script>
</body>
```
