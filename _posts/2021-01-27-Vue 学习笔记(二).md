<p align="center">双向数据绑定指令 v-model</p>
---

v-model 指令一般使用在表单元素上(文本框 多选框 单选框 文本域 下拉框等等)，进行双向数据绑定，一方发生变化，另外一方也会随着改变。v-model 使用在不同的表单元素上，绑定的类型也不一样。

1.v-model 作用于文本框

```
<body>
    <div id="app">
        <div>{ {msg} }</div>
        <input type="text" v-model="msg" />
    </div>
    
    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                msg: 'Hello World'
            }
        })
    </script>
</body>
```

2.v-model 作用于多选框

```
<body>
    <div id="app">
        <div>{ {checked} }</div>
        <input type="checkbox" v-model="checked" />
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                checked: true
            }
        })
    </script>
</body>
```

<p align="center">指令 v-text、v-html</p>
----

v-text 指令用于普通文本插入，将 data 中普通文本内容替换指定元素中的内容。v-html 指令用于 HTML 代码片段插入，将 data 中 HTML 代码片段替换指定元素中的内容。v-text 和 v-html 指令都是单项数据绑定。v-text 指令与插值表达式的区别在于，v-text 指令会将元素中的内容替换成指定数据，插值表达式只会将表达式所在位置替换成指定数据。

1.v-text 指令使用示例

```
<body>
    <div id="app">
        <div v-text="content"></div>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                content: 'Hello World'
            }
        })
    </script>
</body>
```

2.v-html 指令使用示例

```
<body>
    <div id="app">
        <div v-html="content"></div>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                content: '<h1>Hello World</h1>'
            }
        })
    </script>
</body>
```

3.v-text 与插值表达式

```
<body>
    <div id="app">
        <div>---{ {content} }---</div>
        <div v-text="content">------</div>
    </div>
    
    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                content: 'Hello World'
            }
        })
    </script>
</body>
```

<p align="center">属性绑定指令 v-bind</p>
----

v-bind 指令用于属性绑定，将指定的属性与 data 中的数据进行绑定，单项数据绑定。v-bind 指令能够作用于任何属性。该指令的简写形式为 :，使用示例如下：

1.v-bind 指令使用示例

```
<body>
    <div id="app">
        <a v-bind:href="href">百度搜索</a>
    </div>
    
    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                href: 'https://www.baidu.com'
            }
        })
    </script>
</body>
```

2.v-bind 指令简写示例

```
<body>
    <div id="app">
        <a :href="href">百度搜索</a>
    </div>
    
    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                href: 'https://www.baidu.com'
            }
        })
    </script>
</body>
```

v-bind 指令作用于 class 和 style 属性时，vue 做了专门额增强。属性值除了上述示例中属性值是一个字符串映射 data 的属性，还可以是对象或数组，示例如下：

3.v-bind 绑定 class 属性值是字符串

```
<style type="text/css">
    .style1 {
    	background-color: red;
    }
</style>

<body>
    <div id="app">
        <div :class="style">hello world</div>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                style: 'style1'
            }
        })
    </script>
</body>
```

4.v-bind 绑定 class 属性值是个对象

```
<style type="text/css">
    .style1 {
        background-color: green;
    }
    .style2 {
        color: red;
    }
</style>

<body>
    <div id="app">
        <div :class="{ style1 : isStyle1,style2 : isStyle2  }">hello world</div>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                isStyle1: true,
                isStyle2: true
            }
        })
    </script>
</body>
```

5.v-bind 绑定 class 属性值是个数组

```
<style type="text/css">
    .style1 {
        background-color: green;
    }
    .style2 {
        color: red;
    }
</style>

<body>
    <div id="app">
        <div :class="[isStyle1,isStyle2]">hello world</div>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                isStyle1: "style1",
                isStyle2: "style2"
            }
        })
    </script>
</body>
```

6.v-bind 绑定 style 属性值是个对象

```
<body>
    <div id="app">
        <div :style="{ color : colorValue, fontSize : '100px' }">hello world</div>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                colorValue: 'red'
            }
        })
    </script>
</body>
```

<p align="center">事件绑定指令 v-bind</p>
----

1.v-bind 指令使用示例

```
<body>
    <div id="app">
        <button v-on:click="fun">按键</button>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            methods: {
                fun() {
                    console.log('点击事件执行')
                }
            }
        })
    </script>
</body>
```

2.v-bind 指令简写示例

```
<body>
    <div id="app">
        <button @click="fun">按键</button>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            methods: {
                fun() {
                    console.log('点击事件执行')
                }
            }
        })
    </script>
</body>
```

3.方法中 this 指向示例

```
<body>
    <div id="app">
        <button @click="fun">按键</button>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            methods: {
                fun() {
                    // 在方法值 this 指向 vue 示例及 vm，因此可以通过 this 获取 data 中的数据，以及调用 methods 中的方法
                    console.log(this === vm)
                }
            }
        })
    </script>
</body>
```

<p align="center">事件绑定中参数传递、事件对象</p>

----

1.事件方法普通参数传递

```
<body>
    <div id="app">
        <button @click="fun('hello world')">按键</button>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            methods: {
                fun(str) {
                    console.log(str)
                }
            }
        })
    </script>
</body>
```

2.事件方法事件对象参数

```
<body>
    <div id="app">
        // 当有多个参数，事件对象需要使用 $event 传递
        <button @click="fun($event,'hello world')">按键</button>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            methods: {
                // e 为事件对象
                fun(e,str) {
                    console.log(e)
                }
            }
        })
    </script>
</body>
```

3.事件方法没有参数传递

```
<body>
    <div id="app">
        <button @click="fun">按键</button>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            methods: {
                // 事件方法没有参数传递，e 为事件对象
                fun(e) {
                    console.log(e)
                }
            }
        })
    </script>
</body>
```

<p align="center">v-for 指令遍历数据</p>

----

1.遍历数组并展示索引

```
<body>
    <div id="app">
        <ul>
            // item 表示数组的成员，index 为成员对应的索引
            <li v-for="(item,index) in list">{ { item } } { { index } }</li>
        </ul>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                list: ["q","w","e","r"]
            }
        })
    </script>
</body>
```

2.遍历数组不展示索引

```
<body>
    <div id="app">
        <ul>
            <li v-for="item in list">{ { item } }</li>
        </ul>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                list: ["q","w","e","r"]
            }
        })
    </script>
</body>
```

3.遍历对象内部的属性

```
<body>
    <div id="app">
        <ul>
            // item 为属性值，key 为属性名
            <li v-for="(item,key) in user">{ { item } } { {key} }</li>
        </ul>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                user: {
                    name: 'tom',
                    age: 18
                }
            }
        })
    </script>
</body>
```

4.遍历成员为对象数组

```
<body>
    <div id="app">
        <ul>
            <li v-for="item in list">{ { item.name } } { {item.age} }</li>
        </ul>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                list: [
                    {name: 'mary', age: 18},
                    {name: 'jack', age: 18}
                ]
            }
        })
    </script>
</body>
```

5.按指定数子循环遍历

```
<body>
    <div id="app">
        <ul>
            <li v-for="item in 10">{ { item } }</li>
        </ul>
    </div>

    <script src="./vue.js"></script>

        })
```

<p align="center">事件修饰符</p>

----

1.阻止默认行为修饰符

```
<body>
    <div id="app">
        // 跳转指定路径的默认行为被终止，执行方法 fun
        <a href="https://www.baidu.com" @click.prevent="fun">跳转百度</a>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            methods: {
                fun() {
                    console.log('点击跳转百度')
                }
            }
        })
    </script>
</body>
```

2.阻止事件冒泡修饰符

```
<body>
    <div id="app">
        <div @click="fun2">
            // 阻止事件冒泡，只执行 fun1，不执行 fun2
            <button @click.stop="fun1">按键</button>
        </div>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            methods: {
                fun1() {
                    console.log('点击子元素')
                },
                fun2() {
                    console.log('点击父元素')
                }
            }
        })
    </script>
</body>
```

3.关闭冒泡捕获修饰符

```
<body>
    <div id="app">
        // 点击子元素，事件没有被捕获也没有冒泡，只执行 fun1，只有点击父元素才执行 fun2
        <div @click.self="fun2">
            <button @click="fun1">按键</button>
        </div>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            methods: {
                fun1() {
                    console.log('点击子元素')
                },
                fun2() {
                    console.log('点击父元素')
                }
            }
        })
    </script>
</body>
```

4.事件捕获修饰符

```
<body>
    <div id="app">
        //1. 先执行 fun2 再执行 fun1 事件被父元素捕获
        //2. 执行完 fun1 事件没有冒泡，fun2 没有再次执行
        <div @click.capture="fun2">
            <button @click="fun1">按键</button>
        </div>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            methods: {
                fun1() {
                    console.log('点击子元素')
                },
                fun2() {
                    console.log('点击父元素')
                }
            }
        })
    </script>
</body>
```

5.单次触发修饰符

```
<body>
    <div id="app">
        <div @click="fun2">
            // 只有第一次点击子元素 fun1 才会执行，再去点击 fun1 不执行，事件冒充，依旧执行 fun2
            <button @click.once="fun1">按键</button>
        </div>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            methods: {
                fun1() {
                    console.log('点击子元素')
                },
                fun2() {
                    console.log('点击父元素')
                }
            }
        })
    </script>
</body>
```
