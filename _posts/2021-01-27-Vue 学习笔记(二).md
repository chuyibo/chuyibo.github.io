<p align="center">插值表达式</p>
---

```
<body>
    <div id="app">{{content}}</div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                content: 'Hello World'
            }
        })
    </script>
</body>
```

`{{}}`是插值表达式，在插值表达式中可以使用 data 中的数据并展示在页面上，当 data 中的数据发生变化，视图也会发生改变。在插值表达式中可以使用表达式，但是不能使用 if、for、while 语句。
