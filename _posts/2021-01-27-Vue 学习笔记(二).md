<p align="center">双向数据绑定指令 v-model</p>
---

v-model 指令一般使用在表单元素上(文本框 多选框 单选框 文本域 下拉框等等)，进行双向数据绑定，一方发生变化，另外一方也会随着改变。v-model 使用在不同的表单元素上，绑定的类型也不一样。

1.v-model 作用于文本框

```
<body>
    <div id="app">
        <div>{ {msg} }</div>
        <input type="text" v-model="msg" />
    </div>
    
    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                msg: 'Hello World'
            }
        })
    </script>
</body>
```

2.v-model 作用于多选框

```
<body>
    <div id="app">
        <div>{ {checked} }</div>
        <input type="checkbox" v-model="checked" />
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                checked: true
            }
        })
    </script>
</body>
```

<p align="center">指令 v-text、v-html</p>
----

v-text 指令用于普通文本插入，将 data 中普通文本内容替换指定元素中的内容。v-html 指令用于 HTML 代码片段插入，将 data 中 HTML 代码片段替换指定元素中的内容。v-text 和 v-html 指令都是单项数据绑定。v-text 指令与插值表达式的区别在于，v-text 指令会将元素中的内容替换成指定数据，插值表达式只会将替换表达式所在位置替换成指定数据。

1.v-text 指令使用示例

```
<body>
    <div id="app">
        <div v-text="content"></div>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                content: 'Hello World'
            }
        })
    </script>
</body>
```

2.v-html 指令使用示例

```
<body>
    <div id="app">
        <div v-html="content"></div>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                content: '<h1>Hello World</h1>'
            }
        })
    </script>
</body>
```

<p align="center">属性绑定指令 v-bind</p>
----

1.v-bind 指令使用示例

```
<body>
    <div id="app">
        <img v-bind:src="imgSrc">
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                imgSrc: 'https://ss1.bdstatic.com/70cFuXSh_Q1YnxGkpoWK1HF6hhy/it/u=2207157117,1381180556&fm=26&gp=0.jpg'
            }
        })
    </script>
</body>
```
