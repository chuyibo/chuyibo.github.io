```
<body>
    <div id="app">{{content}}</div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                content: 'Hello World'
            }
        })
    </script>
</body>
```

`{{}}`是插值表达式，在插值表达式中可以使用 data 中的数据并展示在页面上，当 data 中的数据发生变化，视图也会发生改变。在插值表达式中可以使用表达式，但是不能使用 if、for、while 语句，示例如下。

1.示例一 

```
<body>
    <div id="app">{{num + 1}}</div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                num: 1
            }
        })
    </script>
</body>
```

> 说明：运算符的使用

2.示例二

```
<body>
    <div id="app">{{ [2,5,7].join('-') }}</div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {}
        })
    </script>
</body>
```

> 说明：方法的调用

3.示例三

```
<body>
    <div id="app">{{ false ? '真' : '假' }}</div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {}
        })
    </script>
</body>
```

> 说明：三元运算符的使用
