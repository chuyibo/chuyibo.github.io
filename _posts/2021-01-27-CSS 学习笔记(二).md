<p align="center">CSS 盒子模型</p>
----

盒子可见框大小由内容区、内边距、边框三部分组成。width 和 height 是设置盒子的内容区大小，而不是盒子的整个大小。background-color 给盒子设置背景色，背景色会延伸至边框。

```
<style type="text/css">
    .box{
        width: 300px;
        height: 300px;
        background-color: red;

        padding: 50px;

        border-width: 10px;
        border-color: blue;
        border-style: dashed;
    }
</style>

<body>
    <div class="box"></div>
</body>
```

![盒子模型](https://chuyibo.github.io/assets/images/007.png)

相邻的垂直方向的外边距会发生外边距的重叠。所谓的外边距重叠指兄弟元素之间的相邻垂直外边距会取最大值而不是取和，如果父子元素的垂直外边距相邻了，则子元素的外边距会设置给父元素。

```
<style type="text/css">
    .box1{
        width: 300px;
        height: 300px;
        background-color: red;

        margin-bottom: 100px;
    }

    .box2{
        width: 300px;
        height: 300px;
        background-color: blue;

        margin-top: 50px;
    }
</style>

<body>
    <div class="box1"></div>
    <div class="box2"></div>
</body>
```

![兄弟盒子垂直外边距重叠](https://chuyibo.github.io/assets/images/008.png)

```
<style type="text/css">
    .box1{
        width: 200px;
        height: 200px;
        background-color: blue;

        margin-top: 50px;
    }

    .box2{
        width: 100px;
        height: 100px;
        background-color: green;

        margin-top: 100px;
    }
</style>

<body>
    <div class="box1">
        <div class="box2"></div>
    </div>
</body>
```
