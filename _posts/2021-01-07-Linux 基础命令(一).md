一、命令格式

----

```
[用户名@主机名 当前目录]命令提示符 命令 [选项] [参数]
```

分隔符`[]`、`@`都没有特殊含义。Linux 系统的默认系统主机名为`localhost.localdomain`，如果不修改默认系统主机名，命令中主机名会显示`localhost`(简写)。如果当前用户为超级用户，命令中命令提示符为`#`，反之命令提示符为`$`。在 Linux 系统中超级用户的家目录为`/root`，非超级用户的家目录为`/home/用户名`。如果当前所在目录为登录用户的家目录，命令中当前目录会显示`~`，反之显示当前所在目录的最后一级子目录。选项用于调整命令的功能，可以省略，参数是命令的操作对象，参数可以省略，在省略的情况下有默认参数。

二、命令示例

----

```
[root@localhost ~]# ls
```

```
[user@localhost ~]$ ls
```

`root`、`user`为用户名，前者为超级用户，后者为普通用户。`localhost` 为主机名。当前登录用户为超级用户，命令提示符为`#`，`~`表示路径`/root`。当前登录用户为普通用户，命令提示符为`$`，`~`表示路径`/home/user`。

```
[root@localhost ~]# cd /boot/grub
[root@localhost grub]# ls
```

切换当前目录为`/boot/grub`，当前所在目录非登录用户的家目录，命令中当前目录显示`grub`，为当前所在目录的最后一级子目录。

```
[root@localhost ~]# ls -a /boot/grub
```

`ls`命令的主要作用是显示目录下的内容，选项`-a`用于调整`ls`命令的功能，表示显示所有目录、文件。参数为`/boot/grub`，表示展示`/boot/grub`目录下的所有目录、文件。如果省略参数，默认参数为当前目录，会展示当前目录下的所有目录、文件。
