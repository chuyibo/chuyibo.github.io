一、命令格式

----

```
[用户名@主机名 当前目录]命令提示符 命令 [选项] [参数]
```

分割符`[]`，没有特殊含义，其功能是将用户信息、主机信息、目录信息与命令提示符进行分割。分隔符`@`没有特殊含义，其左侧为用户名，右侧为主机名。如果当前用户为超级用户，则`[]`中的用户名为`root`。Linux 系统的默认主机名为`localhost.localdomain`，如果不修改默认主机名，则`[]`中的主机名为`localhost`(简写)。

如果当前用户为超级用户，命令提示符为`#`，如果当前用户为普通用户，命令提示符为`$`。在 Linux 中超级用户的家目录为`/root`，普通用户的家目录为`/home/用户名`。如果当前目录为登录用户的家目录，`[]`中的当前目录为`~`，如果当前目录非登录用户的家目录，`[]`中的当前目录为所在目录的最后一级子目录。

选项用于调整命令的功能，可以省略，参数是命令的操作对象，参数可以省略，在省略的情况下有默认参数。

二、命令示例

----

1.示例-登录超级用户

```
[root@localhost ~]# ls
```

2.示例-登录普通用户

```
[chuyibo@localhost ~]$ ls
```

`root`、`chuyibo`为用户名，`localhost` 为主机名。当前登录用户为超级用户，命令提示符为`#`，`~`表示路径`/root`。当前登录用户为普通用户，命令提示符为`$`，`~`表示路径`/home/chuyibo`。

3.示例-切换当前目录

```
[root@localhost ~]# cd /boot/grub
[root@localhost grub]# ls
```

切换当前目录为`/boot/grub`，当前目录非登录用户的家目录，`[]`中的当前目录为`grub`，当前所在目录的最后一级子目录。

4.示例-选项参数使用

```
[root@localhost ~]# ls -a /boot/grub
```

`ls`命令的主要作用是显示目录下的内容，选项`-a`用于调整`ls`命令的功能，表示显示所有目录、文件。参数为`/boot/grub`。整条命令表示展示`/boot/grub`目录下的所有目录、文件。如果省略参数，默认参数为当前目录，会展示当前目录下的所有目录、文件。
