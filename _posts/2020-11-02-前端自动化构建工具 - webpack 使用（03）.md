<pre><div id="code">var path = require('path');
const HtmlWebpackPlugin = require('html-webpack-plugin');
module.exports = {
    entry: './index.js',
    output: {
        path: path.resolve(__dirname, ''),
        filename: 'bundle.js'
    },
    plugins: [
      new HtmlWebpackPlugin({
          template: './index.html'
      })
    ],
    mode: 'none'
};</div></pre>
<p alient="justify">html-webpack-plugin 插件能够将 webpack 构建后所输出的 js 或者 css 样式通过 script 标签或 style 标签引入到指定的 html 中。template 用来指定 html，如果不指定，该插件会创建一个空白的 html 并将构建结果引入到空白的 html 中。</p>
