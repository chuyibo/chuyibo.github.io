<pre><div id="code">div {
    background: blue;
}
</div></pre>

<pre><div id="code">import './index.css'
...
</div></pre>



<pre><div id="code">var path = require('path');
module.exports = {
    entry: './index.js',
    output: {
        path: path.resolve(__dirname, ''),
        filename: 'bundle.js'
    },
    module: {
        rules: [{
            test: /\.css$/,
            use: ['style-loader', 'css-loader']
        }]
    },
    mode: 'none'
};
</div></pre>

<p alient="justify">css-loader 能够解析 css 文件，以字符串的形式存在于 js 文件中，css-loader 的构建结果作为 style-loader 的参数，stype-loader 通过动态生成 stype 标签的形式将字符串样式代码插入到 html 文件中。</p>
