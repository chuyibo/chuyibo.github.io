创建数据库表和与数据库表对应的类，类的属性名与数据库表的字段名一致，类的属性类型与数据库表字段类型对应。表名与类名无需一致。

```
CREATE TABLE user (username VARCHAR(20), password VARCHAR(20));
```

```
public class User implements Serializable {

    private String username;
    private String password;
    省略 Get、Set 方法
}
```

创建持久层接口，基于映射文件的形式需要创建持久层映射文件，持久层映射文件相当于持久层接口的文件。如果基于注解的形式无需创建持久层映射文件，而是在接口方法上使用注解添加实现方法功能的 SQL 语句。

```
public interface IUserDao {

    查询用户信息
    public List<User> selectUserList();
}
```

```
public interface IUserDao {

    查询用户信息，注解添加查询 SQL 语句
    @Select("SELECT * FROM user")
    public List<User> selectUserList();
}
```

创建持久层映射文件，持久层映射文件的名称必须与持久层接口的名称一致，且持久层映射文件与持久层接口必须在相同的包路径下。持久层映射文件相当于持久层接口的实现类。

```
<!-- 持久层映射文件文件名称为 IUserDao.xml -->
<mapper namespace="chu.yi.bo.dao.IUserDao">
    <select id="selectUserList" resultType="chu.yi.bo.domain.User">
        SELECT * FROM user;
    </select>
</mapper>
```

创建 MyBatis 全局配置文件，配置文件名称可任意取名(如 SqlMapConfig.xml)。配置文件包含了影响 MyBatis 行为的设置和属性信息，如数据信息，数据库连接池信息，映射文件信息。

```
<configuration>
    <!--  配置环境 -->
    <environments default="mysql">
        <!-- 配置 mysql 的环境 -->
        <environment id="mysql">
            <!-- 配置事务的类型 -->
            <transactionManager type="JDBC"/>
            <!-- 配置数据源（连接池） -->
            <dataSource type="POOLED">
                <!-- 配置连接数据库的 4 个基本信息 -->
                <property name="driver" value="com.mysql.jdbc.Driver"/>
                <property name="url" value="jdbc:mysql://localhost:3306/test"/>
                <property name="username" value="root"/>
                <property name="password" value="123456"/>
            </dataSource>
        </environment>
    </environments>

    <!-- 指定映射配置文件的位置 -->
    <mappers>
        <mapper resource="chu\yi\bo\dao\IUserDao.xml"/>
    </mappers>
</configuration>
```

根据全局配置文件利用 SqlSessionFactoryBuilder 创建 SqlSessionFactory，使用 SqlSessionFactory 获取 SqlSession，一个 SqlSession 代表和数据库的一次会话。使用 SqlSession 生产持久层接口的代理对象对数据库进行操作。SqlSession 不是线程安全的，每次使用完成后都需要关闭。

```
public void testMyBatis() {
    读取配置文件
    InputStream in = Resources.getResourceAsStream("SqlMapConfig.xml");
    
    创建 SqlSession 对象
    SqlSessionFactoryBuilder builder = new SqlSessionFactoryBuilder();
    SqlSessionFactory factory = builder.build(in);
    SqlSession session = factory.openSession();
    
    使用 SqlSession 创建 dao 接口的代理对象
    IUserDao userDao = session.getMapper(IUserDao.class);
    使用代理对象执行查询操作
    List<User> users = userDao.selectUserList();
    for(User user : users) {
    	System.out.println(user);
    }
    
    释放资源
    session.close();
    in.close();
}
```
