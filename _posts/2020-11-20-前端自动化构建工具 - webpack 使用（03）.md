<pre><div id="code">var path = require('path');
const HtmlWebpackPlugin = require('html-webpack-plugin');
module.exports = {
    entry: './index.js',
    output: {
        path: path.resolve(__dirname, ''),
        filename: 'bundle.js'
    },
    plugins: [
      new HtmlWebpackPlugin({
          template: './index.html'
      })
    ],
    mode: 'none'
};</div></pre>
<p alient="justify">html-webpack-plugin 插件能够将 webpack 的构建结果通过 script 标签引入到指定的 html 文件中，template 属性用来指定 html 文件，如果不指定，该插件会创建一个空白的 html 文件并将 webpack 的构建结果通过 script 标签引入到空白的 html 文件中。</p>
<pre><div id="code">&lt;!DOCTYPE html&gt;

&lt;/head&gt;

&lt;body&gt;&lt;script src=&quot;bundle.js&quot;&gt;&lt;/script&gt;&lt;/body&gt;

&lt;/html&gt;
