<p align="center">基础选择器</p>
----

1.元素选择器

作用：通过元素选择器可以选则页面中的所有指定元素

语法：标签名 {}

```
<style type="text/css">
    p {
    	color: res;
    }
</style>

<body>
    <p>英雄露颖在今朝</p>
    <p>一试矛兮一试刀</p>
    <p>初出便将威力展</p>
    <p>三分好把姓名标</p>
</body>
```

2.id 选择器

作用：通过元素的 id 属性值选中唯一的一个元素

语法：#id 属性值 {}

```
<style type="text/css">
    #poetry {
    	color: res;
    }
</style>

<body>
    <p id="poetry">人情势利古犹今，谁识英雄是白身？</p>
</body>
```

3.类选择器

作用：通过元素的 class 属性值选中一组元素

语法：.class 属性值 {}

```
<style type="text/css">
    .poetry {
    	color: red;
    }
</style>

<body>
    <p class="poetry">威震乾坤第一功</p>
    <p class="poetry">辕门画鼓响冬冬</p>
    <p class="poetry">云长停盏施英勇</p>
    <p class="poetry">酒尚温时斩华雄</p>
</body>
```

4.并集选择器

作用：过选择器分组可以同时选中多个选择器对应的元素

语法：选择器1,选择器2,选择器n {}

```
<style type="text/css">
    h1,.poetry {
    	color: red;
    }
</style>

<body>
    <h1>咏赤壁</h1>
    <p class="poetry">二龙争战决雌雄</p>
    <p class="poetry">赤壁楼船扫地空</p>
    <p class="poetry">烈火张天照云海</p>
    <p class="poetry">周瑜于此破曹公</p>
</body>
```

5.交集选择器

作用：选中同时满足多个选择器的元素

语法：选择器1 选择器2 选择器n {}

```
<style type="text/css">
    div .poetry {
    	color: red;
    }
</style>

<body>
    <p class="poetry">运筹决算有神功，二虎还须逊一龙</p>
</body>
```

6.通配选择器

作用：用来选中页面中的所有的元素

语法：语法：* {}

```
<style type="text/css">
    * {
    	color: red;
    }
</style>
    
<body>
    <h1>念奴娇·赤壁怀古</h1>
    <p>大江东去，浪淘尽，千古风流人物。故垒西边，人道是，三国周郎赤壁。乱石穿空，惊涛拍岸，卷起千堆雪。江山如画，一时多少豪杰。</p>
</body>
```

<p align="center">后代/子元素选择器</p>

----

1.元素之间的关系

```
<div>
    <p>
        <span>温侯神射世间稀</span>
        <span>曾向辕门独解危</span>
        <span>落日果然欺后羿</span>
        <span>号猿直欲胜由基</span>
    </p>
</div>
```

元素之间存在父与子的关系，示例中 div 包含 p，div 和 p 具有父子关系。div 是 p 的父元素，p 是 div 的子元素，同理 p 是 span 的父元素，span 是 p 的子元素。元素之间存在祖先与后代关系，div 包含 p，p 包括 span，div 是 p 和 span 的祖先元素(父元素都是祖先元素)，p 和 span 是 div 的后代元素(子元素都是后代元素)。p 中并列包含多个 span，这些 span 互为兄弟元素。

2.后代元素选择器

作用：选中指定元素的指定后代元素

语法：祖先元素选择器 后代元素选择器 {}

```
<style type="text/css">
    div h1 {
    	color: red;
    }
</style>

<body>
    <div>
        <h1>临江仙</h1>
        <p>滚滚长江东逝水，浪花淘尽英雄。是非成败转头空，青山依旧在，几度夕阳红。</p>
        <p>白发渔樵江渚上，惯看秋月春风。一壶浊酒喜相逢，古今多少事，都付笑谈中。</p>
    </div>
</body>
```

> 说明：示例中选择所有拥有 div 祖先元素的 h1 元素

作用：选中指定父元素的指定子元素

语法：父元素选择器 > 子元素选择器

```
    </div>
</body>
```

> 说明：示例中选择所有拥有 div 父元素的 h1 元素

<p align="center">伪类选择器</p>

----

有些元素具有多种状态，例如超链接有初始状态、被访问状态、鼠标移入状态、被点击状态，文本输入框有失去焦点状态，获取焦点状态等。伪类选择器专门用来选择这些元素的特殊状态，给这些特殊状态设置样式。

1.设置超链接初始状态样式

```
<style type="text/css">
    a:link {
    	color: red;
    }
</style>

<body>
    <div>
        <a href="http://www.baidu.com">古来冲阵扶危主，只有常山赵子龙。</a>
    </div>
</body>
```

2.设置超链接被访问状态样式

浏览器根据历史记录来判断一个连接是否被访问过，由于涉及到用户隐私，所有改选择器只能设置字体颜色。

```
<style type="text/css">
    a:visited {
    	color: red;
    }
</style>

<body>
    <div>
        <a href="http://www.baidu.com">古来冲阵扶危主，只有常山赵子龙。</a>
    </div>
</body>
```

3.设置超链接鼠标移入状态样式

```
<style type="text/css">
    a:hover {
    	color: red;
    }
</style>

<body>
    <div>
        <a href="http://www.baidu.com">古来冲阵扶危主，只有常山赵子龙。</a>
    </div>
</body>
```

4.设置超链接被点击状态样式

```
<style type="text/css">
    a:active {
    	color: red;
    }
</style>

<body>
    <div>
        <a href="http://www.baidu.com">古来冲阵扶危主，只有常山赵子龙。</a>
    </div>
</body>
```

<p align="center">伪元素选择器</p>

----

1.给 p 的第一个字符设置样式

```
<style type="text/css">
    p:first-letter {
    	color: red;
    }
</style>

<body>
    <p>长坂桥头杀气生，横枪立马眼圆睁。</p>
</body>
```

2.给 p 的最后一个字符设置样式

```
<style type="text/css">
    p:first-line {
    	color: red;
    }
</style>

<body>
    <p>长坂桥头杀气生，横枪立马眼圆睁。</p>
</body>
```

3.给 p 前后添加内容并设置样式

```
<style type="text/css">
    p:before{
        content: "“";
        color: red;
    }

    p:after{
        content: "”";
        color: red;
    }
</style>

<body>
    <p>长坂桥头杀气生，横枪立马眼圆睁。</p>
</body>
```

> :first-line 和 :before 一般结合 content 使用。

<p align="center">属性选择器</p>

----

1.作用语法

```
- 作用：可以根据元素中的属性或属性值来选取指定元素
- 语法：
	[属性名] 选取含有指定属性的元素
	[属性名="属性值"] 选取含有指定属性值的元素
	[属性名^="属性值"] 选取属性值以指定内容开头的元素
	[属性名$="属性值"] 选取属性值以指定内容结尾的元素
	[属性名*="属性值"] 选取属性值以包含指定内容的元素
```

2.示例一

```
<style type="text/css">
    [class] {
    	color: red;
    }
</style>

<body>
    <p class="poetry">对酒当歌，人生几何： 譬如朝露，去日苦多。</p>
</body>
```

3.示例二

```
<style type="text/css">
    [class="poetry"] {
    	color: red;
    }
</style>

<body>
    <p class="poetry">对酒当歌，人生几何： 譬如朝露，去日苦多。</p>
</body>
```
