一、常用选择器

----

1.元素选择器

作用：通过元素选择器可以选则页面中的所有指定元素

语法：标签名 {}

```
<style type="text/css">
    p {
    	color: res;
    }
</style>

<body>
	<p>英雄露颖在今朝</p>
	<p>一试矛兮一试刀</p>
	<p>初出便将威力展</p>
	<p>三分好把姓名标</p>
</body>
```
