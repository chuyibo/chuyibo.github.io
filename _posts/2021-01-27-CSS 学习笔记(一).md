<p align="center">CSS 基础选择器</p>

----

1.元素选择器

作用：通过元素选择器可以选则页面中的所有指定元素

语法：标签名 {}

```
<style type="text/css">
    p {
    	color: res;
    }
</style>

<body>
    <p>英雄露颖在今朝</p>
    <p>一试矛兮一试刀</p>
    <p>初出便将威力展</p>
    <p>三分好把姓名标</p>
</body>
```

2.id 选择器

作用：通过元素的 id 属性值选中唯一的一个元素

语法：#id属性值 {}

```
<style type="text/css">
    #poetry {
    	color: res;
    }
</style>

<body>
    <p id="poetry">人情势利古犹今，谁识英雄是白身？</p>
</body>
```

3.类选择器

作用：通过元素的 class 属性值选中一组元素

语法：.class属性值{}

```
<style type="text/css">
    .poetry {
    	color: red;
    }
</style>

<body>
    <p class="poetry">威震乾坤第一功</p>
    <p class="poetry">辕门画鼓响冬冬</p>
    <p class="poetry">云长停盏施英勇</p>
    <p class="poetry">酒尚温时斩华雄</p>
</body>
```

4.并集选择器

作用：过选择器分组可以同时选中多个选择器对应的元素

语法：选择器1,选择器2,选择器n{}

```
<style type="text/css">
    h1,.poetry {
    	color: red;
    }
</style>

<body>
    <h1>咏赤壁</h1>
    <p class="poetry">二龙争战决雌雄</p>
    <p class="poetry">赤壁楼船扫地空</p>
    <p class="poetry">烈火张天照云海</p>
    <p class="poetry">周瑜于此破曹公</p>
</body>
```

5.交集选择器

作用：可以选中同时满足多个选择器的元素

语法：选择器1 选择器2 选择器n{}

```
<style type="text/css">
    div .poetry {
    	color: red;
    }
</style>

<body>
    <div>
        <p class="poetry">运筹决算有神功，二虎还须逊一龙</p>
    </div>
</body>
```

6.通配选择器

作用：他可以用来选中页面中的所有的元素

语法：语法：*{}

```
<style type="text/css">
    * {
    	color: red;
    }
</style>
    
<body>
    <h1>念奴娇·赤壁怀古</h1>
    <p>大江东去，浪淘尽，千古风流人物。故垒西边，人道是，三国周郎赤壁。乱石穿空，惊涛拍岸，卷起千堆雪。江山如画，一时多少豪杰。</p>
</body>
```
