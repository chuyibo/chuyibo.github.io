<p align="center">计算属性</p>

----

计算属性的本质是一个属性，计算属性一定要有返回值。返回结果才是计算属性的最终结果，计算属性可以使用 data 中的数据。计算属性会跟随相关数据的改变而改变，计算属性会重新计算。计算属性不能当方法使用，并且不能跟 data 中的数据重名。

```
<body>
    <div id="app">
        <div>{{add}}</div>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                num1: 10,
                num2: 20
            },
            computed: {
                add() {
                    return this.num1 + this.num2
                }
            }
        })
    </script>
</body>
```

<p align="center">监听器</p>

----

1.监听器监听简单数据

```
<body>
    <div id="app">
        <input type="text" v-model="content"/>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                content: 'hello world'
            },
            watch: {
                content(newVal, oldVal) {
                    console.log("数据变化前",oldVal);
                    console.log("数据变化后",newVal);
                }
            }
        })
    </script>
</body>
```

2.监听复杂数据方式一

```
<body>
    <div id="app">
        <input type="text" v-model="obj.username"/>
        <input type="text" v-model="obj.password"/>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                obj: {
                    username: 'mary',
                    password: '1234'
                }
            },
            watch: {
                obj: {
                    deep: true, // true 表示深度监听
                    immediate: true,  // true 表示立即监听
                    handler(newVal) {
                        console.log(newVal.username);
                        console.log(newVal.password);
                    }
                }
            }
        })
    </script>
</body>
```

3.监听复杂数据方式二

```
<body>
    <div id="app">
        <input type="text" v-model="obj.username"/>
        <input type="text" v-model="obj.password"/>
    </div>

    <script src="./vue.js"></script>
    <script>
        const vm = new Vue({
            el: '#app',
            data: {
                obj: {
                    username: 'mary',
                    password: '1234'
                }
            },
            watch: {
                'obj.username'(newVal) {
                    console.log(newVal)
                },
                'obj.password'(newVal) {
                    console.log(newVal)
                }
            }
        })
    </script>
</body>
```
